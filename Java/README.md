# Platsbokning på flyget

"Platsbokning på flyget" is a flight seat booking app in java.

## Installation

User must have Java 17.

* "Main.java"

* "App.java"

* "SeatPlan.java"

* "Passenger.java"

* "FilePersistence.java"

## User Guide

* Run the program written in "Main.java"

* Follow the on-screen instructions.
